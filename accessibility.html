Performance and Image Optimization

The Lighthouse tool, integrated into the Chrome browser, was used to evaluate our page's performance and accessibility. It recommended the adoption of next-generation image formats for enhanced performance. Consequently, we have converted all our image files to the more efficient webp format.

Accessibility Enhancements

Our page shows commendable adherence to key web accessibility standards, as indicated by the Lighthouse report. This includes ensuring all image elements are equipped with [alt] attributes, essential for screen reader users. Additionally, we have modified the color scheme to improve contrast ratios, making content more readable for users with color vision deficiencies.

Language and Meta Elements

The inclusion of a 'lang' attribute in the HTML element enhances the page's accessibility for language translation tools and screen readers. Also, upon identifying missing meta elements like 'description', we consulted OsloMet's web page for best practices. Subsequently, we have incorporated several meta elements to further enhance our web page's accessibility and SEO optimization.
