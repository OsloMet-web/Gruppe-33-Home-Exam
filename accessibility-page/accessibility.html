Campus Page Accessibility

We have used Lighthouse tool integrated in the chrome browser to test our page for performance and accesibility parameters. Regarding the performance of the page, the tool advised us to use next generation image formats for enhanced performance. So we used an online tool to convert all image files to webp format. Regarding accesibility, the Lighthouse report showed that the page is adhering well to several key web accessibility standards,such as all the image elements are equipped with [alt] attributes, which are crucial for screen reader users. However, we needed to play with the colors that we have chosen to ensure that the background and foreground colors on the page have a sufficient contrast ratio, ensuring readability for users with color vision deficiencies. In the html element we had lang attribute to enhance the page's accessibility for language translation tools and screen readers. The tool warned us that we were missing some meta elements, such as ´description´. We checked OsloMets web page as a guide to use which meta tags. Based on that we added several meta elements to enhance the web pages accesibility and SEO optimization.
